# 子域名信息收集工具

基于 PySide6 的桌面 GUI 工具，用于子域名枚举和信息收集。

## 功能

| 模块 | 说明 |
|---|---|
| crt.sh 被动收集 | 从证书透明日志批量拉取已知子域名 |
| DNS 字典枚举 | 多线程 DNS A 记录爆破（公共 DNS 优先） |
| DNS 记录查询 | A/AAAA/CNAME/MX/NS/TXT/SOA 完整记录 |
| HTTP 探测 + SSL | 状态码 / 标题 / 服务器 / SSL 证书信息 |
| 端口扫描 | ~300 个常见端口，含服务识别 |
| 批量扫描 | 支持逗号分隔或导入文本文件（每行一个域名） |
| 实时保存 | 扫描结果增量写入缓存，崩溃后可恢复 |
| 导出 | JSON（完整字段）/ CSV（表格化） |

## 安装

```bash
# 克隆仓库
git clone https://github.com/wangqiao258/subdomain-scanner.git
cd subdomain-scanner

# 安装依赖
pip install -r requirements.txt


```

## 使用

```bash
python main.py
```

### 扫描流程

1. 输入目标域名（多个用逗号分隔，或点击「导入列表」从文件加载）
2. 选择字典文件（默认使用内置字典）
3. 勾选需要启用的扫描模块
4. 点击「开始扫描」
5. 子域名实时出现在列表中
6. 点击行查看 DNS / HTTP / SSL / 端口 详情
7. 扫描完毕可导出 JSON 或 CSV

### 扫描阶段

```
crt.sh 被动收集  ──→  DNS 字典枚举  ──→  HTTP 探测+SSL  ──→  端口扫描
  阶段1                  阶段2                  阶段3             阶段4
```

各阶段可通过复选框开关。

## 依赖

- Python >= 3.8
- PySide6 — GUI 框架
- dnspython — DNS 查询
- httpx — HTTP 探测
- requests — crt.sh 数据获取

## 项目结构

```
subdomain-scanner/
├── main.py                  # 启动入口
├── requirements.txt
├── data/
│   ├── subdomains.txt       # 内置字典 (~700条)
│   └── subdomains-top1million-110000.txt     # 内置字典 (~110000条)

│   ├── engine.py            # 扫描引擎
│   ├── dns_brute.py         # DNS 字典枚举
│   ├── passive_sources.py   # crt.sh 被动收集
│   ├── dns_records.py       # DNS 记录查询
│   ├── http_probe.py        # HTTP 探测 + SSL
│   └── port_scan.py         # 端口扫描
└── ui/
    ├── main_window.py       # 主窗口 + 实时缓存
    ├── scan_panel.py        # 配置面板
    ├── result_table.py      # 结果表格
    ├── detail_panel.py      # 详情面板
    ├── export_dialog.py     # 导出对话框
    └── resources.py         # 样式表
```
